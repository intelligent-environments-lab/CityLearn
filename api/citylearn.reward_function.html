citylearn.reward_function module

class citylearn.reward_function.IndependentSACReward(agent_count: int, electricity_consumption: Optional[List[float]] = None, **kwargs)[source]

Bases: citylearn.reward_function.RewardFunction

calculate() → List[float][source]

Returned reward assumes that the building-agents act independently of each other, without sharing information through the reward.

Recommended for use with the SAC controllers.

Notes

Reward value is calculated as \([\textrm{max}(-e_0^3, 0), \dots, \textrm{max}(-e_n^3, 0)]\) where \(e\) is electricity_consumption and \(n\) is the number of agents.
